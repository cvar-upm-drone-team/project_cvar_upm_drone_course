# Join the CVAR-UPM Drone Team

## How to Apply

1. Apply through the team website contact form: https://cvar-upm.github.io/drone_team_web/contact/
2. After submitting the form, you will receive an email with a link to this document and the tasks to complete.

## Task Objectives

The goals of these tasks are:

- To evaluate your basic knowledge of the tools we use in the team.
- To make sure you have a working environment ready to develop future work.

## Tasks by Area

If you want to participate in aerial robotics:

- Submit Control Exercise 1: [control/exercises/Exercise1.md](control/exercises/Exercise1.md)
- Submit Control Exercise 2: [control/exercises/Exercise2.md](control/exercises/Exercise2.md)

If you want to participate in computer vision:

- Submit Vision Exercise 1: [vision/exercises/Exercise1.md](vision/exercises/Exercise1.md)
- Submit Vision Exercise 2: [vision/exercises/Exercise2.md](vision/exercises/Exercise2.md)

If you are not sure yet:

- Submit Exercise 1 and Exercise 2 from both areas, then decide which one you want to join.

For PhD positions, submit also Exercise 3:

- Control Exercise 3: [control/exercises/Exercise3.md](control/exercises/Exercise3.md)
- Vision Exercise 3: [vision/exercises/Exercise3.md](vision/exercises/Exercise3.md)

## How to Submit

Send an email to `cvar.upm.racing@gmail.com` with:

- A link to your repository with the completed tasks uploaded, or
- A `.zip` file of this repository with your completed tasks.
- All extra deliverables required by each exercise (for example, videos or method descriptions), included in the email.

Use this email subject format:

`Submit Drone Course - Control/Vision/Control&Vision Name Surname`
